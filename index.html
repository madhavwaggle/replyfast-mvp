Integration Setup Guide

Follow these steps to connect Zillow, Homes.com, and SMS to your live ReplyFast instance.

1

Deploy to Vercel

Push this project to GitHub, then import it at vercel.com. Vercel will auto-detect and deploy the backend API routes in the /api folder.
# In your terminal: git init git add . git commit -m "Initial ReplyFast deploy" git remote add origin https://github.com/YOUR_USERNAME/replyfast.git git push -u origin main
2

Set Vercel environment variables

In your Vercel project → Settings → Environment Variables, add these:

ANTHROPIC_API_KEYYour key from console.anthropic.com
TWILIO_ACCOUNT_SIDFrom twilio.com console
TWILIO_AUTH_TOKENFrom twilio.com console
TWILIO_PHONE_NUMBERYour Twilio number e.g. +15135550100
POSTMARK_SERVER_TOKENFrom postmarkapp.com → Server → API Tokens
AGENT_EMAILYour email for lead notifications
3

Set up Postmark inbound email (for Zillow / Homes.com)

1. Create a free account at postmarkapp.com
2. Go to your Server → Inbound → you'll get an address like xyz@inbound.postmarkapp.com
3. Set the Inbound Webhook URL to: https://YOUR-APP.vercel.app/api/inbound-email
4. In Zillow Premier Agent or Homes.com, set your lead notification email to that Postmark address.
5. Every new lead email gets parsed and enters your dashboard automatically within seconds.
4

Set up Twilio SMS

1. Sign up at twilio.com — you get $15 free credit
2. Buy a local phone number (~$1/month)
3. Set the number's Incoming Message Webhook to: https://YOUR-APP.vercel.app/api/inbound-sms
4. Leads who text your Twilio number will be captured and AI will respond within 60 seconds.
5. Give agents this number as their "text for info" number on yard signs.
5

Connect your website contact form

Add this to any HTML form's submit handler to send leads directly to ReplyFast:
// Replace with your deployed Vercel URL fetch('https://YOUR-APP.vercel.app/api/new-lead', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ fname, lname, email, phone, property, message, source: 'Website' }) })
